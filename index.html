I am a PhD candidate in the Computer Science Department at the College of William and Mary, advised by Dr. Denys Poshyvanyk. I am also a member of the SEMERU Research Group. Before joining SEMERU, I worked in the field of program verification with Dr. Isil Dillig. I received a MCS degree from Miami University, where I was advised by Dr. Keith Frikken, and a B.S. degree with honors from Shandong Normal University.

I have also been working with Dr. Nicholas A. Kraft, Dr. David C. Shepherd, and Dr. Vinay Augustine since May 2015 as a visiting scientist in software engineering group at corporate research center of ABB. The goal of our projects is to help developers enhance both efficiency and accuracy of their works.

My research interests include software evolution, source code summarization and Human-Computer Interaction.

Projects

  • Block-based robot programming - an UI design project for blocky and visual programming on industrial robotics.
  • Flowlight is a physical light that reduces working interruptions. The work was featured on New Scientist,  The Telegraph, GeekWire, NBC News, New Atlas, DigitalTrends, Business Standard, The New Yorker, TechXplore, MailOnline/DailyMail, ScienceDaily, News For Everyone, Evening Express, Yahoo News, India Today, PPP Focus, The Statesman, Radio Canada, BT, Telengana Today, Le Matin (French), CnBeta (Chinese), PopMech (Russian), PcNews (Russian), Teknikan Maailma (Finnish), Utusan (Malaysian), Irish Examiner, Tech.Rizlys, Appsforpcdaily.com, EurekAlert, Lancashire Post, MetroNews, Breaking News, UBC News, UBC Science, and many other blogs.
  • ChangeScribe.NET is a visual studio plug-in for generating commit messages automatically. The approach uses as input two versions of a project, and generates a message with a general description of the changes and a detailed description of each change in natural language.
  • DBScribe is a tool for generating up-to-date natural language descriptions of database operations and schema constraints in source code methods.
  • DACITE helps users to find conflicts between source code and database by using symbolic execution and associative rule mining.
  • PISTIS (stands for Protecting and mInimizing databases for Software TestIng taSks) uses a weight-based data clustering algorithm that partitions data in the database using information from program analysis that indicate how this data is used by the application. [site]
  • HOLA (stands for HOare Logic with Abduction) is an inductive invariant generation tool based on logical abduction.
